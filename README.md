# Accounting Engine

ON GOING PORJECT

Backend accounting system built with Node.js, TypeScript, Prisma, and Redis.

## Features
- Authentication system (Register, Login, Logout, Refresh Token)
- JWT Authentication
- Redis-based rate limiting
- Integration Tests with Vitest

## Tech Stack
- Node.js
- TypeScript
- Prisma
- PostgreSQL
- Redis
- Vitest

## Setup

```bash
npm install
npm run dev
```

Run Tests
```bash
npm run test
